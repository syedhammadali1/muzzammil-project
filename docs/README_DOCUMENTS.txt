INSTRUCTIONS FOR CREATING WORD DOCUMENTS
=========================================

The detailed school information is provided in .txt format for easy copy-paste.

To create the .docx files:

1. Open Microsoft Word (or any word processor)
2. Copy the content from each .txt file
3. Paste into a new Word document
4. Format the document with:
   - Headings and subheadings
   - Bold text for section titles
   - Proper spacing
   - Page numbers (optional)
5. Save as .docx format with the filename:
   - greenfield_international_school.docx
   - sunshine_public_school.docx
   - riverside_academy.docx
   - maplewood_elementary.docx
   - oakridge_high_school.docx
   - meadowbrook_school.docx

The .txt files contain all the content needed. Simply format and save as .docx files.

Alternatively, you can use online converters or scripts to batch convert .txt to .docx.
